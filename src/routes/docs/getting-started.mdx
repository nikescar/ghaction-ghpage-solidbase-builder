# Getting Started

Welcome to Jekyll Solidbase Theme! This guide will help you get up and running quickly.

## Installation

1. Add this theme to your Jekyll site's `_config.yml`:

```yaml
remote_theme: nikescar/jekyll-theme-solidbase
```

2. Install the required dependencies:

```bash
bundle install
```

## Basic Configuration

Add the following to your `_config.yml`:

```yaml
theme_settings:
  en:
    title: Your Site Title
    nav:
      - Docs: /docs
    sidebar:
      - Docs:
          - Getting Started: /docs/getting-started
          - Features: /docs/features
```
